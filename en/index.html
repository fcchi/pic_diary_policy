Privacy Policy

Last updated: February 2025

1. Information We Collect

This app stores diary entries and AI-generated images in cloud storage (Firebase Firestore, Cloud Storage).

① User-Created Diary Data

  • User diary entries are stored in Firebase Firestore.
  • Diary data is linked to anonymous login UIDs and can only be viewed, edited, or deleted by the user.
  • Developers and third parties cannot access or use diary data.

② AI-Generated Image Data

  • Images generated by AI based on user diary entries are stored in Cloud Storage.
  • Stored images can only be viewed and downloaded by the user.
  • Image data is not shared with developers or third parties.
  • Users can delete images within the app.

③ Advertisement-Related Information

  • This app uses Google AdMob to display advertisements.
  • AdMob collects Advertising IDs and device information to optimize ad delivery.

2. Data Usage Purpose

  • User diary data and generated images are stored as core app functionality.
  • No one other than the user can access this data.
  • Advertisement data is used for app maintenance and improvement.

3. User Choices

  • Users can delete diary data and images within the app.
  • To turn off ad personalization, visit Ad Settings.
